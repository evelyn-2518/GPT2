## ✅ 三、Markdown 快速語法（常用）

| 語法 | 說明 |
|------|------|
| `#` / `##` / `###` | 標題（第1～3層） |
| `**粗體**` | 粗體 |
| `*斜體*` | 斜體 |
| `` `code` `` | 行內程式碼 |
| \`\`\`python<br>程式碼區塊<br>\`\`\` | 多行程式碼 |
| `- 項目` | 清單 |
| `[超連結文字](網址)` | 超連結 |
| `![圖片文字](圖片網址)` | 插入圖片 |
