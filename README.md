# 專題自習 GPT-2中文補全
## Environment
- Python 3.10
- CUDA 12.4
- Ubuntu 22.04 
## Requirements
-  [`pyproject`](https://github.com/evelyn-2518/GPT2/blob/main/pyproject.toml) 
## 開始
```bash
# 從 GitHub 複製專案
git clone https://github.com/evelyn-2518/GPT2.git
cd GPT2

# 安裝所需套件
pip install -r pyproject.toml
```
